About Pyp-boy

Pyp-boy is a text RPG made with Python and Flask, inspired by Fallout's Pip-boy layout.

This software has been created by Seb and is licensed under MIT.

The story has been written by Seb and is licensed under CC-BY-SA 4.0.

Please see the License declaration available on the application's repository.

This software includes AI-generated images using Craiyon, and the story initially written in French has been translated using DeepL and a manual review. The maps are based on OpenStreetMap, using Leaflet and Stamenmaps rendering.Some assets are coming from the original Fallout franchise and are included under the fair use term.

Please refer to the Attribution section of the repository's README.
